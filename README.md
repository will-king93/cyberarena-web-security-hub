## What CyberArena Is

CyberArena is a web-only training environment for anyone learning cybersecurity and ethical hacking. It emphasizes doing rather than just reading, with guided quizzes, game-like challenges, and a simulated terminal that introduces common Linux and hacking commands in an approachable format.

The project is built with HTML, CSS, and JavaScript, and it runs fully in the browser without a backend. As a result, it is simple to open and share, works across modern devices, and keeps the focus on interactive practice.

## How to Use It

Open CyberArena in a browser and pick the learning mode that matches what you want to study.

Typical workflow:
1. Begin with a quiz level to refresh core cybersecurity ideas.
2. Continue into the interactive games for a more hands-on challenge.
3. Try the command line simulator to work with Linux and hacking-style commands.
4. Revisit lessons whenever you want more repetition and confidence.

Since the project is front-end only, interaction happens through the UI instead of server-side commands or an API workflow.
